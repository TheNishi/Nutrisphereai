# 🥗 NutriSphereAI — AI-Powered Fitness & Nutrition Tracker

## 🚀 Overview

NutriSphereAI is a modern, AI-driven web application designed to make fitness tracking simple, smart, and engaging. It helps users track their daily calorie intake, analyze meals through image recognition, and receive personalized health recommendations.

This project combines **web development + AI + UI/UX design** to create a real-world, impactful health-tech solution.

---

## ✨ Features

### 🍽️ AI Food Recognition

* Upload or capture food images
* Detect food items using AI
* Get instant nutritional details:

  * Calories 🔥
  * Protein 💪
  * Carbohydrates 🍞
  * Fats 🥑

---

### 📊 Smart Dashboard

* Daily calorie tracking
* Calories consumed vs remaining
* Macronutrient breakdown (Protein, Carbs, Fat)
* Interactive circular progress UI

---

### 🥞 Meal Management

* Track:

  * Breakfast
  * Lunch
  * Dinner
  * Snacks
* Add food manually or via AI scan
* View meal-wise calorie distribution

---

### 🧠 AI Recommendations

* Personalized suggestions based on:

  * Fitness goals (Weight Loss / Gain / Maintain)
  * Daily intake
* Example:

  * “You need 20g more protein today”

---

### 👤 User Profile

* Input:

  * Age, Height, Weight
  * Fitness Goal
* BMI Calculation
* Daily calorie goal estimation

---

### 🏋️ Workout Tracker

* Suggest exercises
* Track calories burned
* Maintain activity logs

---

### 🌙 Additional Features

* Dark Mode Toggle
* Responsive Design (Mobile + Desktop)
* Smooth animations & modern UI
* Optional AI chatbot assistant

---

## 🛠️ Tech Stack

### Frontend

* HTML

### Backend

* Flask / Node.js

### AI Integration

* TensorFlow / Computer Vision APIs
* Nutrition APIs (Edamam / Spoonacular)

## 📸 How It Works

1. Upload a food image
2. AI detects the food item
3. Nutritional data is extracted
4. Data is added to your daily dashboard
5. Get smart recommendations

## 📦 Installation

```bash
git clone https://github.com/your-username/nutrisphereai.git
cd nutrisphereai
```

### Run Frontend

```bash
open index.html
```

### Run Backend (Flask example)

```bash
pip install -r requirements.txt
python app.py
```
